# DD Image

DD Image is a c++ application for search and delete duplicate images.

## Supported formats:

* [jpg]
* [png]
## Installation

```bash
mkdir build
cd buid
cmake ..
make
```
## Usage

```bash
./DDImage /path_to_folder_whit_images
```
